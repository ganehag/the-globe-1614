# The Globe: 1614 - A One-Page TTRPG

## Overview
*The Globe: 1614* is a one-page tabletop role-playing game (TTRPG) set in the historical context of The Globe Theatre in the year 1614. Players take on the roles of second and third-rate actors tasked with performing a crucial play under challenging circumstances. The game emphasizes improvisation, creativity, and the whimsical nature of theater.

This repository contains the LaTeX source code for the game document, including an English version (`the-globe-1614.tex`) and a Swedish version (`the-globe-1614-sv.tex`).

## Project Structure
- `src/the-globe-1614.tex`: The English version of the game document.
- `src/the-globe-1614-sv.tex`: The Swedish version of the game document.
- `README.md`: This file, providing an overview and instructions.
- `LICENSE`: License information for the project.

## How to Compile
To compile the LaTeX documents into PDF format, follow these steps:

1. Ensure you have a LaTeX distribution installed (e.g., TeX Live, MikTeX).
2. Navigate to the project directory in your terminal.
3. Run the following command to compile the English version:

   ```bash
   pdflatex src/the-globe-1614.tex
   ```
4. For the Swedish version, run:

   ```bash
   pdflatex src/the-globe-1614-sv.tex
   ```
